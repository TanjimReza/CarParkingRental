
## Run Locally

Clone the project

```bash
  git clone https://github.com/TanjimReza/CarParkingRental.git .
```

Go to the project directory

```bash
  cd CarParkingRental
```

Install dependencies

```bash
  pip install django 
  pip install -r requirements.txt 
```

Start the server

```bash
  cd CarParkingRental
  python manage.py runserver
```
